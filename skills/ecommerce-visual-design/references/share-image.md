# 分享图设计规范（站外引流）

分享图用于站外社交/私域引流，`1:1:2K` 或 `3:4:2K`（按目标平台）。目标：引发好奇→点进商品。

## 视觉工艺要点

- **社交钩子**：疑问/对比/利益前置，引发好奇（如「这件事居然……」「对比后我选了它」）。
- **弱广告感**：比主图更接近 UGC/种草感但仍保持商业品质；避免一眼硬广。
- **主体突出**：产品或使用结果是视觉中心；文字 1 钩子句 + 1 利益点，不超 2 行。
- **平台口吻**：按 `target_platform`/分享渠道调整——小红书偏生活方式图文感、微信偏信任背书、抖音偏强冲击。
- **合规**：不夸大、无极限词；引导语符合平台规则（避免诱导分享违规）。
- **一致性**：产品与主图/详情同一件；可复用锚点参考或共享 `$STYLE`。
