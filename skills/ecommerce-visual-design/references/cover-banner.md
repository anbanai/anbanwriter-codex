# 封面 / 类目 Banner 设计规范（品牌氛围与入口）

封面/类目 banner 承担店铺/活动/类目入口的视觉定调，`16:9:2K` 或 `3:4:2K`。

## 类型与目标

- **店铺/品牌 banner**：品牌氛围 + 主推品 + slogan，定调品牌质感。
- **类目 banner**：类目名称 + 代表品 + 导购暗示，引导进入类目。
- **活动 banner**：活动主题 + 利益点（满减/赠品）+ 时间，合规真实。

## 视觉工艺要点

- **氛围优先**：宽幅构图，留白与场景感强于主图；弱化密集信息。
- **品牌一致性**：延续 `$STYLE` 与产品锚点；logo/品牌色规范统一。
- **文字精简**：1 主标题（≤10 字）+ 1 副标题/利益点；字号大、留白足。
- **主体位置**：产品主体偏左/右三分之一，文字避让；符合三分法构图。
- **移动/PC 双适配**：16:9 适合 PC/类目顶图，3:4 适合移动端竖版入口——按投放位选尺寸。
- **禁**：极限词、与活动不符的虚假利益、遮挡主体的文字。
